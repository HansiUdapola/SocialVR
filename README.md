# SocialVR
This project is to test the Ubiq Network library for Social VR, which is developed by the University College of London. Please visit their site here if you want to integrate their networking library for your Unity Project. ðŸ‘‰ [![Ref Link](https://img.shields.io/badge/-Ubiq%20Online-green?logo=html5&logoColor=white)](https://ubiq.online/) [![Ref Link](https://img.shields.io/badge/-Ubiq-blue?logo=github)](https://github.com/UCL-VR/ubiq)

Key functionalities:
------------------
By referring to their demo project, I created a new VR environment and added different ways to change the avatar within the virtual space. This was also tested with UMA by integrating with the Unity Multipurpose Avatar system (UMA).
